# Trip Logger App

A simple yet powerful Android application that helps you track travel expenses. Built entirely with modern Kotlin and Jetpack Compose, it offers management trips and their associated costs.

## Features

- [x] Log multiple trips  
- [x] Add and view expenses for each trip  
- [x] View a running total of all expenses  
- [x] Edit existing expenses (UI prepared, logic planned)  
- [x] Delete expenses (UI prepared, logic planned)

## Tech Stack

- Android (Kotlin, Jetpack Compose)
- Architecture: MVVM
- Storage: Room (local persistence)
- Navigation: Compose Navigation
- Async: Kotlin Coroutines & Flow
- UI Guidelines: Material Design 3
